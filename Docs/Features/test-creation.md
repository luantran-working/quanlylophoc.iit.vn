# Tạo bài kiểm tra (Test Creation)

## Tổng quan

Tính năng Tạo bài kiểm tra cho phép giáo viên tạo các bài kiểm tra trắc nghiệm online, gửi đến học sinh và tự động chấm điểm.

## Quy trình chi tiết

### 1. Tạo bài kiểm tra

```
┌─────────────────────────────────────────────────────────────┐
│                    TẠO BÀI KIỂM TRA                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. Click "Tạo bài kiểm tra" trong sidebar                 │
│                      │                                      │
│                      ▼                                      │
│  2. Nhập thông tin cơ bản:                                 │
│     - Tên bài kiểm tra                                      │
│     - Môn học                                               │
│     - Thời gian làm bài                                     │
│                      │                                      │
│                      ▼                                      │
│  3. Thêm câu hỏi:                                          │
│     ┌────────────────────────────────────────┐              │
│     │ a. Click "Thêm câu hỏi"                │              │
│     │ b. Nhập nội dung câu hỏi               │              │
│     │ c. Nhập 4 đáp án (A, B, C, D)          │              │
│     │ d. Chọn đáp án đúng (Radio button)     │              │
│     │ e. Lặp lại cho các câu khác            │              │
│     └────────────────────────────────────────┘              │
│                      │                                      │
│                      ▼                                      │
│  4. Cấu hình cài đặt:                                      │
│     - Xáo trộn câu hỏi/đáp án                               │
│     - Hiển thị đáp án sau khi nộp                           │
│     - Cho phép làm lại                                      │
│                      │                                      │
│                      ▼                                      │
│  5. Click "Gửi bài kiểm tra"                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. Gửi bài kiểm tra

```
┌─────────────────────────────────────────────────────────────┐
│                    GỬI BÀI KIỂM TRA                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. Sau khi click "Gửi bài kiểm tra"                       │
│                      │                                      │
│                      ▼                                      │
│  2. Popup chọn học sinh:                                   │
│     ┌────────────────────────────────────────┐              │
│     │  Chọn học sinh nhận bài kiểm tra:      │              │
│     │                                        │              │
│     │  ☑ Tất cả học sinh online             │              │
│     │  ☐ Chọn từng học sinh                 │              │
│     │                                        │              │
│     │         [Gửi] [Hủy]                   │              │
│     └────────────────────────────────────────┘              │
│                      │                                      │
│                      ▼                                      │
│  3. Bài kiểm tra được gửi đến các máy học sinh             │
│                      │                                      │
│                      ▼                                      │
│  4. Học sinh nhận thông báo popup                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3. Học sinh làm bài

```
┌─────────────────────────────────────────────────────────────┐
│                     LÀM BÀI (Học sinh)                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. Nhận thông báo từ giáo viên:                           │
│     ┌────────────────────────────────────────┐              │
│     │  📝 Bài kiểm tra mới!                  │              │
│     │                                        │              │
│     │  Kiểm tra 15 phút - Chương 1           │              │
│     │  Thời gian: 15 phút                    │              │
│     │  Số câu: 10 câu                        │              │
│     │                                        │              │
│     │       [Bắt đầu làm bài]               │              │
│     └────────────────────────────────────────┘              │
│                      │                                      │
│                      ▼                                      │
│  2. Click "Bắt đầu làm bài"                                │
│                      │                                      │
│                      ▼                                      │
│  3. Cửa sổ làm bài mở ra:                                  │
│     - Đồng hồ đếm ngược                                     │
│     - Danh sách câu hỏi                                     │
│     - Các đáp án để chọn                                    │
│                      │                                      │
│                      ▼                                      │
│  4. Trả lời từng câu hỏi (Radio button)                    │
│                      │                                      │
│                      ▼                                      │
│  5. Click "Nộp bài" khi xong                               │
│     (hoặc tự động nộp khi hết giờ)                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Loại câu hỏi

| Loại            | Mô tả                            |
| --------------- | -------------------------------- |
| Multiple Choice | Chọn 1 đáp án đúng từ 4 lựa chọn |
| True/False      | Đúng hoặc Sai                    |
| Fill in Blank   | Điền vào chỗ trống (tương lai)   |

## Best Practices

1. **Câu hỏi rõ ràng**: Tránh câu hỏi mơ hồ
2. **Đáp án hợp lý**: Các đáp án sai phải có vẻ hợp lý
3. **Thời gian phù hợp**: 1-2 phút/câu
4. **Xáo trộn**: Bật xáo trộn để tránh gian lận
5. **Preview**: Xem thử trước khi gửi
